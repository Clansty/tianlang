# QYBot C# 插件 SDK

## 版本

当前分支的版本为: v1.0 preview 预览版

## 特色

- 支持解析回复信息及其他以二进制传送的信息

- 在内部对框架传入的参数信息进行预处理，再通过参数的形式传递给供用户编辑的方法，使你的代码更整洁

- 对 QY 框架的 API 进行封装，并重写所有注释，使用更方便

## 用法

1. 下载 master 分支的源码

2. 通过 Visual Studio 打开解决方案，并打开 `QYEvents.cs` 文件

3. 找到 `PluginInfo` 类，填写插件基本信息

4. 编辑 `QYEvents` 类中需要的方法，开始插件的编写
